# Jadwal Shalat

Aplikasi sederhana untuk menampilkan jadwal shalat harian.

## Fitur

- Menampilkan jadwal shalat untuk hari ini
- Data diambil dari file JSON
- Tampilan responsif

## Cara Penggunaan

Buka `index.html` di browser Anda untuk melihat jadwal shalat.
